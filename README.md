# Pointiv

Context-aware AI desktop layer for macOS.

## Installation

1. Download the latest `Pointiv_x.x.x_aarch64.dmg` from the releases page
2. Open the DMG and drag **Pointiv.app** to your Applications folder
3. Run this command in Terminal (required on macOS — the app is not yet notarized):

```bash
xattr -cr /Applications/Pointiv.app
```

4. Open Pointiv from Applications

## Shortcut

Default trigger: **⌘⇧Space** — select any text, hit the shortcut, and Pointiv appears.
